## 💻 Projeto

Aplicação de calendário da Copa do Mundo 2022.

## ✨ Tecnologia

- [React](https://www.w3schools.com/html/)
- [ViteJS](https://www.w3schools.com/css/default.asp)

## 🚀 Como executar

Para executar em ambiente de desenvolvimento, rode os seguintes comandos:

```bash
cd todo-list
yarn dev
```
